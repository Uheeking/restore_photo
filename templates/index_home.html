ImageDenoising

Hold the camera for a few seconds to get a frame of the same scene. Then find the average of all frames and compare the first frame with the last frame to remove the noise. This method is called denoising. It takes more time compared to the blurring technique we saw earlier, but the result is very good.

Convolution blurring

This method is used in a famous algorithm called cnn in deep learning. It is a simple principle that is widely used. To keep the pixel values ​​level, they are divided by equal values. Putting the blur into the kernel with the same values ​​is called average blurring.

Gaussian Denoising

For each matrix, a kernel having a Gaussian distribution can be inserted, which is called Gaussian blurring. Gaussian distribution is another name for a normal distribution, and to easily understand, it is a distribution with the highest median value and fewer toward the end.

Bilateral filter

If Gaussian Blur is applied heavily, the edges in the image will be damaged. To overcome this shortcoming, a technique called bidirectional filters has emerged. It gets its name from the fact that the Gaussian filter is applied twice in both directions. The bidirectional filter adjusts the degree of blurring by considering the distance between the reference pixel and the neighboring pixel and the difference between the pixel values.

Median blurring

Unlike Gaussian blurring, the difference is that one pixel in the kernel area is selected and put as it is. In other words, if the result is obtained by literally averaging pixel values ​​ (or similar operation) through conventional blurring, the median is a concept that selects the pixel value corresponding to the median value.
